WritePad Editor

WritePad is an editor application allows you to create Web pages easily and quickly.

Steps to open WhitePad application:

  • Click Start
  • Click Programs
  • Click WhitePad

Advantages of using a WhitePad editor:

  • It allows faster creation of Web pages.
  • It allows easier updation of text or images in WhtePad editor.
  • It is platform independent.

Disavantages of using a WritePad editor:

  • It consumes more memory space.
  • It requires to fill-in more details such as meta data for each page.
